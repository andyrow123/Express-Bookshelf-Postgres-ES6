## ES6-ExpressJS-Starter
ES6 with Grunt, Babel, ExpressJS, SASS &amp; BrowserSync.


### Main Components:
 - Grunt 
 - Babel Compiler
 - ExpressJS
 - SASS
 - BrowserSync


### Usage:

```sh
# clone a copy
git clone https://github.com/kurtisdunn/ES6-ExpressJS-Starter.git
cd ES6-ExpressJS-Starter

# get it going:
npm install

# run it
node dist/server.js

```

```sh

## Development Stuff

# Compile ES6 to ES5
grunt compile

# run up a development version:
grunt dev

```
